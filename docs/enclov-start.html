enclov start command

Starts the Enclov AI CLI application. This command initializes your environment and sets up the connection with the Enclov backend services.

Usage

enclov start [options]

Options

  • --env <environment> Specify the environment to start (e.g., production, staging, dev).
  • --verbose Enable verbose logging output.

Example

enclov start --env production --verbose

Notes

This command must be run from the project root directory.
